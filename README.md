## Installation

1. Clone this repo:

```bash
git clone https://github.com/srikrishna-ps/decentralized-intelligence.git
cd decentralized-intelligence
```

2. Install dependencies:

```bash
npm install
```

3. Compile contracts:

```bash
npx hardhat compile
```
